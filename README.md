# Joplin Attrs Markdown plugin

This plugin adds attrs capability as in markdownIt plugin [attrs plug-in](https://github.com/arve0/markdown-it-attrs).

## Usage

``` markdown
# header {..c-red}
```

output
```html
<h1 css-module="c-red">header</h1>
```

build in color attribute
c-red
c-yellow
c-blue
c-green
c-pink
c-purple
c-lightyellow
c-lightblue
c-lightgreen
c-lightpink
